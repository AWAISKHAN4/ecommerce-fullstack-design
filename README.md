# Ecommerce Full-Stack Design

A modern full-stack eCommerce web application built with React, Node.js, Express.js, and MongoDB. The project includes user authentication, product management, shopping cart functionality, and a responsive user interface.

##  Live Demo

**Frontend:** https://ecommerce-fullstack-design-xi-ten.vercel.app

**Backend API:** https://ecommerce-fullstack-design-i0fp.onrender.com

##  GitHub Repository

https://github.com/AWAISKHAN4/ecommerce-fullstack-design

##  Features

- User Authentication (JWT)
- Product Listing
- Product Details
- Shopping Cart
- Search Products
- Admin Panel
- Responsive Design
- MongoDB Database
- REST API

##  Tech Stack

### Frontend
- React.js
- Vite
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
